# DTube — Youtube без лишних отвлекающих элементов

**DTube** — это минималистичный веб-интерфейс для работы с материалами на YouTube, созданный для борьбы с цифровой зависимостью и "дофаминовыми ловушками" современных платформ.

Проект полностью исключает алгоритмические рекомендации, бесконечные ленты (Shorts), комментарии и всплывающие уведомления, оставляя пользователя наедине с контентом и инструментами продуктивности.

---

## Основные функции

*    **Чистый поиск:** Интеграция с официальным **YouTube Data API v3**. Поиск возвращает ровно 6 релевантных видео — никаких бесконечных прокруток.
*    **Помодоро-таймер:** Встроенный таймер (Час фокуса / 5 минут отдыха) помогает держать ритм учебы прямо во время просмотра лекции.
*    **Заметки:** Модуль для ведения конспектов в реальном времени. Все заметки сохраняются локально в браузере.
*    **Безопасность данных (Privacy First):** API-ключ пользователя и все его заметки хранятся строго локально в `localStorage` вашего браузера. Никаких внешних баз данных или сбора метрик.
*    **Modern UI:** Минималистичная темная тема, сверстанная на **Tailwind CSS**.
